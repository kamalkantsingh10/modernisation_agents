# Glossary Template — COBOL Field Name to Business Term Mapping

Use this file to map COBOL field/variable names to human-readable business terms
for a new client engagement. One row per mapping.

Copy this file to `<engagement-name>/glossary.md` and populate it.

## Instructions

- **COBOL Name**: the exact field name as it appears in COBOL source (uppercase, hyphens)
- **Business Term**: the plain-English business meaning understood by the client
- Add one row per field. Do not change the table header format.

## Mappings

| COBOL Name         | Business Term               |
|--------------------|-----------------------------|
| WS-CUST-BAL        | Customer Account Balance     |
| WS-ACCT-NO         | Account Number               |
| WS-TXN-DATE        | Transaction Date             |
| WS-PROD-CODE       | Product Code                 |

# BlackJack Demo Engagement — Glossary

Field-name-to-business-term mappings for the BlackJack COBOL demonstration corpus.
Source: https://github.com/kamalkantsingh10/cobol-blackjack

| COBOL Name              | Business Term                  |
|-------------------------|--------------------------------|
| WS-PLAYER-HAND          | Player Card Hand               |
| WS-DEALER-HAND          | Dealer Card Hand               |
| WS-CARD-VALUE           | Card Face Value                |
| WS-ACE-HIGH             | Ace Counted as High (11)       |
| WS-ACE-LOW              | Ace Counted as Low (1)         |
| WS-HAND-TOTAL           | Hand Point Total               |
| WS-DECK-COUNT           | Remaining Cards in Deck        |
| WS-BET-AMOUNT           | Player Bet Amount              |
| WS-PLAYER-BALANCE       | Player Chip Balance            |
| WS-GAME-RESULT          | Round Outcome (Win/Lose/Draw)  |
| WS-BUST-FLAG            | Hand Exceeds 21 Flag           |
| WS-STAND-FLAG           | Player Standing Flag           |
| WS-AUDIT-TRAIL          | Game Audit Record              |
| WS-DECK                 | Card Deck State                |
| WS-GAME-STATUS          | Current Game Phase             |
